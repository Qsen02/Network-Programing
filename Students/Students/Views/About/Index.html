About

This application demonstrates the: Create, Read, Update and Delete (CRUD) operations.
It is developed using Model-View-Controller (MVC) software architectural pattern.
Implemented web server provides services using the Hypertext Transfer Protocol (HTTP) methods: GET, POST, UPDATE and DELETE.
